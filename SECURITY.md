# Security Policy

## Supported Versions

This is a portfolio framework. Security fixes are accepted for the current
`master` branch.

## Reporting a Vulnerability

Please do not open a public issue for sensitive security concerns. Instead,
contact the repository owner directly through GitHub.

If you are reporting a dependency issue, include:

- The affected package and version
- The vulnerability identifier, if available
- The suggested fixed version
- Any known impact on this project
